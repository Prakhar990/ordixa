# Ordixa

Ordixa is a lightweight academic order management platform designed for small-scale operations.

## Features
- Customer order placement & tracking
- Admin approval system with audit logs
- Writer dashboard for task handling
- WhatsApp-based payment coordination
- Exportable records (JSON / CSV ready)

## Tech Stack
- Node.js
- Express.js
- Vanilla HTML, CSS, JavaScript
- File-based JSON storage

## Use Case
Ideal for hostels, institutes, or small academic teams managing assignment workflows.
